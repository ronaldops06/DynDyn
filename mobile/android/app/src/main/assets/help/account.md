## Contas

São contas correntes, poupanças, carteiras digitais, dinheiro físico (cash), investimentos e outros tipos de contas. 

Nesta tela você pode acompanhar o saldo atual, com base nas suas transações, de cada conta.
Organize-as em uma hierarquia personalizada para visualizar o total consolidado com clareza.

### Acessar tela de criação

Para criar uma conta **clique no botão "+"**, posicionado no lado inferior direito da tela, isto irá abrir a tela para criação.

### Acessar tela de edição

Se for necessário editar uma conta, **dê um clique** nela e assim será aberta a tela para edição.

### Alterar status

Clique e **arraste a conta para a esquerda** para alterar o status atual (ativa/inativa).

### Excluir

Clique e **arraste a conta para a direita** para excluir.

> Caso a conta a ser excluída tenha contas filhas ou esteja associada a alguma transação, não será possível excluir.
> Mas será exibida uma mensagem de alerta informando a situação.
